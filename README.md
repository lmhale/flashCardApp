# Description
An app where a user can make flashcards to learn coding concepts.

## Features

Basic Card Operations

- Create a card
- Read a card
- Delete a card
- Update a card

- Flip a card
- Automatic check(for capturing responses)
- Mark a card correct or incorrect

- Organized by Decks
- Tags
- Correct or incorrect
- Syntax vs concept

## Stretch Features
Spaced repetition
